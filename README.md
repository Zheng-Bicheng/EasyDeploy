# EasyDeployForRK3568

## 简介

**EasyDeployForRK3568**是针对柠檬派RK3568开发的推理部署工具箱。EasyDeploy集成了多种AI模型，能快速的在柠檬派RK3568上部署各种例程，
做到开箱即用，大大缩减了工程所需要的时间。

## 目前支持的模型列表



| 模型类型      | 模型名称    | 是否量化 | 编程语言   |
|-----------|---------|------|--------|
| Clas      | AdaFace | 否    | python |
| Detection | ScrFD   | 否    | python |
