## 安装EasyDeploy

### python

**Install EasyDeploy Python For PC**
```text
git clone https://github.com/Zheng-Bicheng/EasyDeploy
python setup.py install
```

**Install EasyDeploy Python For Board**
```text
git clone https://github.com/Zheng-Bicheng/EasyDeploy
sudo python3 setup.py install
```
